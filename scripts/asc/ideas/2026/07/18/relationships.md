- Links (synonym : edges ?)
- Bridges (synonym : associations ?)
- Software
- Hardware

Examples :

- pipe is 1 type of bridge, as well as stdin / stdout / stderr redirection wrappers (*-wrap)
- io (synonmys : emitter / reciever, origin / destination, start / end ...)
