# ASC core concept : Documentation

Kinds of documentation :

- changelog/* : done (archived) or active (in progress) planified modifications + minimal tracking
- docs/* : living documentation files used to explain everything there is to know about the project.
- READMEs = "what is this folder / project ... about" - why it exists (function), capabilities, organization, usage, etc. : idem, but can be contextual (nested dir = nested README)
